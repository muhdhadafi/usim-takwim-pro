# 📅 USIM Tamhidi Academic Calendar

An interactive academic calendar for USIM Tamhidi students, parents, lecturers, and staff.

🌐 Live Website:
https://usim-takwim-pro.vercel.app/

## Features

- Monthly calendar view
- Academic timeline
- Semester schedules
- Examination schedules
- Public holidays
- Dark mode
- Mobile responsive
- Progressive Web App (PWA)
- Offline support

## Disclaimer

This is an independent community project and is not affiliated with Universiti Sains Islam Malaysia (USIM). Please refer to official USIM announcements for the latest academic information.
